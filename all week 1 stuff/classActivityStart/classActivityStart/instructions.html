WDD 230 Lesson 1 Class Activity

For the class activity, you have several choices. Below, the requirements for three small programs are listed. With your group, choose one of them to start. If you complete it, select another one and start working on it. See how much you can accomplish during the class period. You may choose from the following:

This will be a great exercise that incorporates html, css and JavaScript.

  • Make a bulleted list of 10 items.
  • Below the list have an input field and button that will add new items to this list when submitted.
  • Style every other bullet in the list.
  • Make it look nice :)
  • Everything should be responsive for mobile, tablet and desktop screens
  • If you don't know how to do everything it is ok try your best!
